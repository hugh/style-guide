Design System Reference

Retro Game

A pixelated design system inspired by 8-bit and 16-bit era video games — chunky pixel fonts, health bar gauges, character select cards, and arcade continue screens for single-page HTML documents.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and thick pixel-art borders.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Retro Game system.

1

CRT Blue-Black

Backgrounds use cool blue-tinted near-blacks like the glow of a powered-on CRT monitor in a dark arcade.

2

Player Green

The primary accent is a vivid 8-bit green inspired by health bars, "GO" signals, and player-one indicators.

3

Game-World Semantics

Four semantic colors map to game concepts: health green (HP/safe), coin gold (treasure/loot), fire red (damage/danger), magic blue (mana/shields).

4

Chunky Pixel Type

Press Start 2P delivers authentic 8-bit title screen typography for headings and labels. VT323 gives body text a readable retro terminal feel.

5

Zero Border-Radius

No rounded corners anywhere. Everything is sharp and blocky like pixel art, reinforcing the retro digital aesthetic.

6

Scanline Overlay

A faint repeating horizontal line texture across the entire page simulates a CRT monitor, tying all elements together.

7

Game UI Components

Health bars, character select cards, achievement badges, and continue screens bring video game HUD elements into the design language.
