00 — what's different

Why "Rough"?

The original woodworking theme is clean and systematic. This version aims to feel more hand-made. Here's what changed:

🔨

Uneven Warmth

Cards aren't all the same color. Each one pulls from a slightly different warm tone, like different boards from the same tree.

✏️

Handwritten Voice

Margin notes in Caveat. Struck-through text. Amendments. The content reads like a person's actual notebook.

📐

Slight Rotations

Callouts, stamps, and tags rotate a degree or two. Not everything sits perfectly square on the page.

🌲

Rougher Textures

Heavier wood grain, thicker borders on some elements, more visible grid on blueprints. The "grain" is coarser.

The goal isn't chaos — it's warmth. A human made this, not a machine.

05 — principles

What Makes It "Rough"

1

Varied Card Warmth

Five --warm-N tokens get applied via nth-child, so cards look like different boards from the same tree.

2

Handwritten Layer

Caveat font for margin notes, inline scrawl, and principle numbers. This is the "pencil on the plans" layer.

3

Slight Rotations

Callouts, stamps, tags, and swatches all rotate slightly. Nothing dramatic — 0.3° to 3°. Just enough to feel human.

4

Imperfect Spacing

Section padding doesn't match top to bottom. Odd sections get thicker borders. The container is 880px instead of 900.

5

Personal Content

The template reads like someone's actual build notes, not a generic placeholder. Opinions, mistakes, crossed-out text.

6

No Animation

The polished version has smooth fade-ups. This one doesn't. Things just appear, like they've been there all along.

The rough version isn't trying to look bad. It's trying to look like a person made it instead of a system.
