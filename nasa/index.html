Houston // Design System Reference

Mission Control

Clean, confident, engineering-precise. A design system inspired by 1960s–70s NASA mission control — telemetry readouts, countdown timers, CAPCOM logs, and mission patch badges for single-page HTML documents.

Getting Started

Copy template.html and tokens.css into your project. The template includes every component ready to customize. The stylesheet has all variables, resets, and component styles.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically like flight controller console stations in the MOCR.

Max-width 900px, centered, with 2rem horizontal padding.

⚙️

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace telemetry label in the format 01 — Label, followed by an h2 heading, then body content and components.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Mission Control system.

1

Deep Navy, Not Black

Backgrounds use blue-tinted dark tones reminiscent of the MOCR at night, not pure black. This creates the feel of instrument panel glow in a darkened room.

2

GO/NO-GO Semantics

Four semantic colors map directly to mission control console indicators: GO green (nominal), caution amber (advisory), abort red (critical), flight blue (procedural).

3

Engineering Precision

IBM Plex Mono for all telemetry data, timestamps, and technical readouts. Every number is crisp, monospaced, and instantly scannable.

4

Clean Geometric Headings

Space Grotesk delivers confident, modern geometric headings that echo the clean engineering aesthetic of NASA’s worm logo era.

5

Instrument Panel Layout

Telemetry grids, countdown displays, and CAPCOM logs bring the spatial organization of a real flight controller console into the design.

6

Mission Identity

Mission patch badges provide a strong visual anchor, echoing the tradition of unique crew patches for every spaceflight since Gemini.

7

Subtle Grid Texture

The hero section uses a faint grid overlay evoking the graph-paper precision of trajectory plotting boards and console CRT displays.
