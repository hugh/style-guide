Cyberpunk

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and subtle separators against the rain-soaked dark backdrop.

📏

Container

Max-width 900px, centered, with 2rem horizontal padding.

⚙️

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace label in the format 01 — Label, followed by an h2 heading, then body content and components.

Glitch Block

A callout/card with a corrupted-data aesthetic: scan-line texture overlay and a CSS glitch animation that triggers on hover, briefly shifting the element via clip-path and transform. A glitching horizontal bar sweeps across the block during the animation.

█ Corrupted Data Stream Source: Unknown Integrity: 12%

Signal fragmented. The glitch block uses a scan-line texture via ::before and a glitch bar via ::after. On hover, the block triggers a clip-path + transform shift animation to simulate data corruption. The left border uses --capture (electric cyan).

Holographic Card

A translucent card with a multi-color shimmer gradient that shifts on hover. The background uses background-size: 200% 200% and animates background-position on interaction. A holographic sweep overlay animates continuously via ::before.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Cyberpunk / Neon system.

1

Rain-Soaked Black, Not Pure Black

Backgrounds use purple-tinted near-blacks like wet pavement at night, not pure #000. The purple undertone reflects the neon light pollution of the cityscape.

2

Neon Glow is Key

The primary visual effect is neon luminance: multi-layered text-shadow on featured text, box-shadow blur on card borders, and radial-gradient ambient light on containers.

3

Glitch & Corruption

Data decay and signal interference are core aesthetic elements. Scan lines, clip-path glitch shifts, and corrupted data headers reinforce the cyberpunk atmosphere.

4

Holographic Shimmer

Multi-color translucent gradients with animated position shifts create the illusion of holographic projection surfaces and AR overlays.

5

Monospace Everything

JetBrains Mono for body text and IBM Plex Mono for labels give the entire document a terminal/hacker aesthetic, as if rendered on a CRT or neural implant display.

6

Geometric Futuristic Headings

Orbitron delivers sharp, angular uppercase headings that evoke corporate megastructure signage and cybernetic interface chrome.

7

Sharp Edges, Minimal Radius

Border-radius of 2px throughout creates a hard, angular, tech-industrial aesthetic. No soft curves — everything is razor-cut and precise.
