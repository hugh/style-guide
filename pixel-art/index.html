06 — Key Patterns

Design Principles

1

4-Color Constraint

Everything renders in exactly 4 shades of green-grey, recreating the Game Boy DMG-01 screen. No additional colors.

2

Pixel-Sharp Edges

Border-radius is always 0px. Every corner is pixel-perfect and square, matching the bitmap rendering of the hardware.

3

8px Grid System

All spacing, sizes, and alignment follow an 8px base grid — the tile size used by Game Boy hardware for sprites and backgrounds.

4

Pixel Font Headings

Press Start 2P provides authentic bitmap-style typography for headings and labels. Inter handles body text readability.

5

Thick Pixel Borders

2-4px solid borders give everything a chunky, hardware-rendered feel. Dashed and dotted variants mimic screen artifacts.

6

Game UI Components

Dialogue boxes, sprite cards, save points, and HP bars bring video game interface patterns into document design.
