Design System Reference

Kaiju

A dark, ominous design system inspired by Japanese monster movie posters — radiation-green accents, hazard stripes, emergency broadcasts, and seismograph dividers for single-page HTML documents.

v0.1 · view starter template

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and subtle separators on the dark background.

📏

Container

Max-width 900px, centered, with 2rem horizontal padding.

⚙️

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace technical label in the format 01 — Label, followed by an h2 heading, then body content and components.

05 — Kaiju Components

Warning Stripe

A header bar wrapped in diagonal yellow/black hazard stripes. Content sits inside a dark inner panel. Use for high-priority warnings and restricted zones.

⚠ Hazard Zone Detected

The warning stripe uses a repeating-linear-gradient at 45deg to create classic construction/hazard tape diagonals. An inner panel provides readable contrast.

Emergency Broadcast

A red-bordered alert box with a pulsing left border, styled like an Emergency Broadcast System message. The header uses monospace technical lettering.

█ Emergency Broadcast System

This is not a drill. A creature of unknown origin has been detected approaching the coastline. All residents within the evacuation zone must proceed to designated shelters immediately. The pulsing border uses a CSS animation cycling between --execute and --execute-dim.

Seismograph Divider

An inline SVG waveform that mimics a seismograph readout. Flat baseline with periodic spikes of activity. Use between content sections for thematic separation.

Threat Level

A horizontal DEFCON-style indicator with five escalating segments. Add the inactive class to dim unselected levels. Each level uses a different semantic color.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Kaiju system.

1

Ash & Smoke, Not Black

Backgrounds use warm-tinted near-blacks like smoke and ash, not pure #000. This creates depth reminiscent of a destroyed cityscape at night.

2

Radiation Glow

The primary accent is a vivid radiation green with box-shadow glows on interactive elements. It evokes nuclear energy and mutation.

3

Monster-Movie Semantics

Four semantic colors map to monster-movie concepts: radiation green (mutation), plasma orange (fire), destruction red (danger), deep sea blue (ocean depths).

4

Military/Technical Labels

Share Tech Mono gives section labels, tags, and metadata the feel of classified military readouts and emergency broadcast systems.

5

Bold Condensed Headings

Anton delivers ultra-condensed uppercase headings that echo vintage Japanese monster movie poster typography.

6

Hazard Patterns

Warning stripes, emergency broadcasts, and seismograph dividers bring real-world emergency infrastructure into the design language.

7

Scanline Texture

The hero section uses repeating horizontal scanlines and faint crack textures, evoking old CRT monitors displaying monster-tracking footage.
