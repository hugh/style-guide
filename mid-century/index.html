Design System Reference

Mid-Century Modern

A warm atomic-age design system — mustard and teal accents, starburst dividers, boomerang shapes, and Googie-inspired panels for single-page HTML documents.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing against the warm cream background.

📏

Container

Max-width 900px, centered, with 2rem horizontal padding.

⚙️

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace label in the format 01 — Label, followed by an h2 heading, then body content and components.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Mid-Century Modern system.

1

Warm Cream Backgrounds

Light backgrounds use warm off-white and cream tones, creating an inviting retro canvas that evokes Danish modern interiors.

2

Mustard Gold Accent

The primary accent is a rich mustard gold that was ubiquitous in 1950s–60s interiors. It marks headings, callout borders, and principle numbers.

3

Atomic-Age Semantics

Four semantic colors map to the mid-century palette: teal (fresh/modern), mustard (warm/craft), coral (bold/alert), slate blue (cool/info).

4

Generous Rounded Corners

12px border-radius on all cards and panels, with pill-shaped 20px radius on tags, giving the system a soft, organic mid-century feel.

5

Bold Geometric Headings

Poppins delivers clean, geometric headings with tight letter-spacing that channel the Futura-adjacent type of the atomic age.

6

Retro Unique Components

Starburst dividers, boomerang cards, Googie panels, and atomic badges bring iconic mid-century motifs into the design language.

7

Atomic Starburst Hero

The hero section features a subtle repeating-conic-gradient starburst and a faint boomerang accent shape, setting the retro tone immediately.
