Design System Reference

The Western

A sun-bleached parchment design system inspired by the Wild West — wanted poster cards, sheriff badge markers, branding-iron dividers, and saloon-door panels for single-page HTML documents.

v0.1 · view starter template

Getting Started

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and rope-line borders between each territory.

📜

Container

Max-width 900px, centered, with 2rem horizontal padding.

📌

Sections

5rem vertical padding. Separated by 2px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace label in the format 01 — Label, followed by an h2 heading, then body content and components.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Western system.

1

Sun-Bleached Parchment

Backgrounds use warm sandy tones like weathered wanted posters nailed to a post, baking in the desert sun.

2

Burnt Sienna Accent

The primary accent evokes branding irons, leather, and desert heat — the core color of the frontier.

3

Three-Tier Text

Dark wood (--text), worn ink (--text-dim), and faded marking (--text-faint) create clear typographic hierarchy.

4

Saloon Display Headings

Rye delivers the ornate, Old West signage typography of frontier saloons and general stores.

5

Clean Body Serif

Lora provides readable body text like a well-set territorial gazette or frontier newspaper.

6

Frontier Semantics

Four semantic colors: cactus green (frontier), dusty gold (treasure), wanted red (danger), denim blue (sky).

7

Western Artifacts

Wanted posters, sheriff badges, branding dividers, and saloon panels bring the frontier into the design language.

8

Woodgrain Texture

Repeating horizontal lines in the hero evoke weathered plank boards of a frontier town storefront.
