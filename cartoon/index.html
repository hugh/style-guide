Design System Reference

Cartoon Network

A candy-colored design system inspired by animated TV shows — thick rounded outlines, bubbly oversized type, bouncy hover animations, speech bubbles, action badges, and episode guide cards for single-page HTML documents.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and thick cartoon outlines.

💻

Container

Max-width 900px, centered, with 2rem horizontal padding.

📌

Sections

5rem vertical padding. Separated by 3px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace label in the format 01 — Label, followed by an h2 heading, then body content and components.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Cartoon system.

1

Bright White Base

Backgrounds use clean white like animation cels, letting candy colors pop against thick dark outlines.

2

Hot Pink Accent

The primary accent channels Powerpuff Girls / Teen Titans Go energy — playful, not epic.

3

Candy-Color Semantics

Four vivid candy colors: green (go), orange (star), red (pow), blue (cool). Full saturation, maximum fun.

4

Bubbly Rounded Type

Fredoka delivers oversized, rounded, playful heading energy. Nunito keeps body text warm and friendly.

5

Thick Cartoon Outlines

3px dark borders on everything create the signature animated look — bold, confident, unmissable.

6

Big Rounded Corners

16px border-radius on cards and panels. 24px on speech bubbles. 20px on pill tags. Everything is bubbly.

7

Bouncy Animations

Hover transforms use cubic-bezier(0.34, 1.56, 0.64, 1) for playful overshoot, like a bouncing cartoon character.

8

Toon Artifacts

Speech bubbles, action badges, character cards, and episode guides bring the animated TV show into the design language.
