00 — what changed

Rough vs Rougher

The rough-woodworking/ theme added handwritten annotations, card warmth variation, and slight rotations. This rougher version turns all those knobs further and adds new physical-artifact classes: tape, coffee rings, pushpins, dog-ears.

04 — new rougher components

Coffee Ring

Add .coffee-ring to any positioned element. A faint brown ring appears in the top-right corner — like someone set down a mug on the plans.

A Card With a Coffee Ring

The ring is an 80px radial-gradient circle, very low opacity. It's decorative only — doesn't affect layout. The ring is positioned via ::after so the element needs position: relative (or any positioned value).

Masking Tape

Add .taped to stick a small masking tape strip across the top of any element. Uses ::before with a clip-path for rough edges.

A Taped-On Note

The tape strip is about 90px wide, slightly off-white and transparent (uses --tape token), rotated -2.5°. It sits on top of the card, extending above it by a few pixels.

Pushpin

Add .pinned for a small red pushpin dot in the top-left corner. Uses ::before.

A Pinned Card

The pushpin is a 12px circle with a subtle gradient and drop shadow. It looks like someone tacked this card to a corkboard.

Dog-Ear

Add .dog-ear for a folded corner triangle in the top-right. Uses ::after with a CSS triangle. Note: the element needs overflow: hidden (which the class applies).

A Dog-Eared Card

The triangle is 24px. The fold color matches --bg so it looks like the corner has been bent back to reveal the background behind the card.

Pencil Underline

Add .pencil-underline to any inline text for a wavy underline effect in the accent color.

Here is some text with a pencil underline applied to it and then the sentence continues normally. Good for drawing attention to specific words.

Combining Classes

These classes can be combined. Here's a card that's taped, coffee-ringed, and dog-eared:

The Full Treatment

Tape on top, coffee ring in the corner, dog-ear fold. This is what a real workshop note looks like after a week on the bench.

Don't use all of these on every element. Pick one or two. This demo is intentionally over the top.

Margin Notes (Louder)

Margin notes are bigger now (1.2rem), more opaque (0.9), and intrude into the content area with a small negative left margin.

This note is harder to ignore than the Level 1 version. It practically elbows its way into the paragraph. The border is thicker too — 3px instead of 2px.

Inline: scrawl text is 1.2rem now instead of 1.15rem. And you can mark things as wrong corrected with the same utility classes.

Lumber Stamps (Wilder Rotation, Double-Border)

Stamps rotate 3–6° now (was 0.5–3°). Some are more faded (40% vs 55%). New .double-border variant adds an outline.

Pseudo-element conflicts: .coffee-ring and .dog-ear both use ::after. .taped and .pinned both use ::before. Combining two classes that use the same pseudo-element will cause one to override the other. Plan accordingly.
