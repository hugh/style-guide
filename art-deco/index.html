Design System Reference

Art Deco

A 1920s Gatsby-era design system with burnished gold accents, geometric ornaments, jewel-tone semantics, and Art Deco typography for single-page HTML documents.

v0.1 · view starter template
— 00 — Quick Start

Getting Started

Copy template.html and tokens.css into your project. The template includes every component ready to customize. The stylesheet has all variables, resets, and component styles.

◆

template.html

Starter document with every component wired up
◇

tokens.css

Drop-in stylesheet — variables, reset, all components

Usage: Link tokens.css in your HTML head and use the class names documented below. No build step required.

— 03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing.

◆

Container

Max-width 900px, centered, with 2rem horizontal padding.

◇

Sections

6rem vertical padding. Separated by 1px border-top in --border color.

◊

Full-Bleed

Diagrams can break out to 1100px max-width using the .full-bleed class.

◈

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a Josefin Sans label in the format — 01 — Label, followed by an h2 heading, then body content and components.

Gilded Frame

A double-bordered card with gold corner chevrons. Use for featured content, pull quotes, or key callouts. Styled with box-shadow: inset for the stepped border effect.

Featured Content

The gilded frame uses a 1px gold outer border, a 3px surface-colored inset shadow, then a 1px border-colored inner line. Corner chevrons are rendered with ::before and ::after pseudo-elements.

Deco Divider

A geometric ornamental separator. The diamond is set as text content of the element, with gold lines extending via pseudo-elements.

— 05 — Key Patterns

Design Principles

These patterns define the visual identity of the system.

1

Dark with Golden Warmth

Backgrounds are not pure black or gray — they carry a subtle warm, golden undertone across all surface levels.

2

Geometric Precision

Sharp corners, no border-radius on cards or containers. Diamond and chevron motifs replace circles and soft curves.

3

Gold Ornamental Accents

Burnished gold is used sparingly for borders, dividers, and emphasis. The deco divider, gilded frame, and monogram provide ornamental richness.

4

Three-Tier Text Hierarchy

Cream/ivory primary (--text) for headings and emphasis. Dim (--text-dim) for body paragraphs. Faint (--text-faint) for labels and metadata.

5

Wide Letterspacing

Headings use 0.08em tracking. Labels and tags use 0.3em. This creates the characteristic Art Deco typographic rhythm.

6

Jewel-Tone Semantics

Four named color categories — Emerald, Amber, Ruby, Sapphire — each with full, dim, and transparent-fill variants. Cards, pipeline stages, tags, and diagrams all reference these consistently.
