Design System Reference

Board Game

A warm, tactile design system inspired by tabletop board games — felt-green surfaces, wooden token accents, player cards, dice rolls, and score tracks for single-page HTML documents.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and subtle separators on the green felt background.

📏

Container

Max-width 900px, centered, with 2rem horizontal padding.

⚙️

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace label in the format 01 — Label, followed by an h2 heading, then body content and components.

📖Placement Rulesp. 12

Tiles must be placed adjacent to at least one existing tile on the board. Matching edges score bonus points. A tile cannot be placed if it would create an impossible configuration.

Example: Placing a forest tile next to a river tile scores 2 bonus points. A castle tile adjacent to 3 or more roads scores 5 bonus points.
⚠Penaltyp. 24

If a player cannot place a tile on their turn, they lose 3 points and draw a new tile. The skipped tile is returned to the bottom of the draw pile.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Board Game system.

1

Felt-Green Table

Backgrounds use warm forest greens like the felt surface of a gaming table, creating an inviting, tactile foundation.

2

Wooden Token Gold

The primary accent is a warm gold inspired by natural wood game pieces, tokens, and meeples.

3

Game-Night Semantics

Four semantic colors map to tabletop concepts: roll green (success), token gold (score), penalty red (danger), meeple blue (player).

4

Rounded & Friendly

Nunito Extra Bold and Quicksand deliver warm, approachable typography with generous rounding. 8-10px border-radius throughout reinforces the tactile feel.

5

Physical Components

Player cards, dice, score tracks, and rulebook callouts bring the physicality of board games into the design language.

6

Board Grid

Faint grid lines in the hero evoke the squares and hexes of a game board surface.

7

Playful Yet Structured

The design balances warmth and fun with clear structure — monospace metadata, consistent spacing, and organized information hierarchy keep things readable.
