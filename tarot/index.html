Tarot

A mystical occult design system — midnight indigo surfaces, gold-foil accents, celestial star fields, tarot card frames, crystal ball callouts, and zodiac markers for single-page HTML documents.

03 — Layout

Structure

Documents use a single-column layout with a centered container. Sections stack vertically with generous spacing and subtle separators against the midnight indigo background.

📏

Container

Max-width 900px, centered, with 2rem horizontal padding.

☽

Sections

5rem vertical padding. Separated by 1px solid border-top.

↔️

Full-Bleed

Diagrams can break out to 1100px max-width with the full-bleed class.

📱

Responsive

Fluid hero type with clamp(). Card grids collapse to 1 column below 640px.

Section Pattern

Every section follows the same structure: a monospace numeral label in the format 01 — Label, followed by an h2 heading, then body content and components.

06 — Key Patterns

Design Principles

These patterns define the visual identity of the Tarot / Occult system.

1

Midnight, Not Black

Backgrounds use deep indigo and purple-tinted near-blacks, never pure #000. This creates the velvet depth of a candlelit reading room.

2

Gold Foil Accents

The primary accent is a warm gold (#d4a830) evoking gilded tarot card edges, gold-leaf illuminated manuscripts, and candlelit metallic surfaces.

3

Occult Semantics

Four semantic colors map to arcane concepts: emerald (life/growth), gold (divination/insight), blood ruby (danger/death), sapphire (wisdom/the cosmos).

4

Arcane Typography

Cinzel Decorative gives headings an ornamental, woodcut-engraved quality. It evokes ancient manuscripts, occult grimoires, and carved stone inscriptions.

5

Clean Serif Body

Crimson Pro provides elegant, readable body text that balances the ornate headings with classical serif refinement.

6

Celestial Star Fields

The hero section uses scattered radial-gradient dots and ambient celestial glows to create a star-field texture, as if the document floats in the night sky.

7

Ornate Double Borders

Tarot cards and zodiac markers use layered borders (outer bold, inner fine) to create the ornate frame aesthetic of traditional tarot decks.
