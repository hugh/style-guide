SEC 02 — Typography

Font Stack

Three IBM/technical fonts. Condensed uppercase for headings, IBM Plex Sans for body, IBM Plex Mono for data.

IBM Plex Sans — Body text, card content, descriptions (weights 300–600)

IBM Plex Mono — Labels, tags, table headers, metadata, code

SEC 05 — Design Principles

Key Patterns

01

Blueprint Grid Background

Two-scale grid overlay on the body (20px minor, 100px major) gives every page the feel of graph paper or a drafting surface.

02

Sharp Corners Everywhere

Zero border-radius on all components — cards, pipeline stages, diagram containers, tags, callouts. Rectilinear geometry only.

03

Condensed Uppercase Headings

Barlow Condensed with uppercase + letter-spacing for all headings. Mirrors the labeling style on engineering drawings.

04

Status-Driven Color System

Five colors mapping to workflow states: line (working), redline (revision), approved (complete), caution (warning), note (annotation).

05

Title Blocks & Drawing Numbers

Diagram containers include a lower-right title block. Cards use drawing-number style labels (SPEC-001, DWG-002). Section labels use SEC prefixes.

06

No Decoration, Maximum Precision

No shadows, no hover animations, no rounded corners, no gradients on components. Clean lines and flat surfaces — like a technical drawing.
